# ToDoリスト アプリ

シンプルなToDo管理アプリです。

## 機能

- タスクの追加（Enterキーまたは「追加」ボタン）
- 完了チェック（丸ボタンをクリック）
- タスクの削除（ホバーで表示される × ボタン）
- ローカルストレージによるデータ永続化

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## Vercelへのデプロイ

1. [Vercel](https://vercel.com) にサインアップ
2. GitHubにこのプロジェクトをpush
3. VercelでGitHubリポジトリをインポート
4. 自動デプロイが完了

または Vercel CLI を使用:

```bash
npm i -g vercel
vercel
```
